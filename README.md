# red-nosed-report

## Requisiti  

- **Rust**

## Installation  

1. Clone the repository:  
   ```bash
   git clone https://github.com/tuo-utente/red-nosed-report.git
   cd red-nosed-report
   ```  

2. Complie the project:  
   ```bash
   cargo build --release
   ```  

## Usage  

Exec the program with  
```bash
./target/release/red-nosed-report
```
